# Farmasi Manager - Requerimientos

## 1. Contexto
Control de ventas de productos Farmasi.
- **Precios Variables**: Costo fijo, Precio Venta variable.
- **Usuarios**: 3 (Admin, Vendedor).

## 2. Filosofía
- Simple, Visual, Rápido ("Mobile-first").
- Tipo "Treinta" (Claridad financiera).
- Prioridad: **Ganancia Real**.

## 3. Módulos
1. **Dashboard**: Ingresos, Ganancia, Stock Bajo.
2. **Productos**: Costo, Precio Sugerido, Stock.
3. **Inventario**: Kardex simple.
4. **Ventas (POS)**:
   - Seleccionar productos.
   - **Editable**: Precio venta final.
   - Cálculo automático de ganancia.
5. **Gastos**: Registro simple.
6. **Reportes**: Exportables.

## 4. Stack Técnico
- Framework: Next.js (React)
- Estilo: Vanilla CSS (Premium Dark Mode)
- Iconos: Lucide React
- Datos: Local JSON (Prototipo) / Base de datos futura.
